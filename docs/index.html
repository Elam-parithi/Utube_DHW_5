About the Project

This product was writen by Elamparithi. This project focuses on Harvesting and Analyzing YouTube data with Google YouTube API module, MySQL DB, Mongo DB. Aditionaly, Here were are using sentiment Analaysis on comments. It provides insights into Channel Playlists ,videos, comments and its sentiment using NLTK.

Github project was available on github.com/Elam-parithi/Utube_DHW_5

Key Features

  • Data extraction from YouTube API
  • Storing that data in JSON
  • Comment sentiment analysis
  • Channel and video analytics
  • SQL and MongoDB integration

# Config page:

In config page we provide the API key from google website, and SQL and MongoDB URL and URI. When you click on the connect button if passed the status bar will turn the color to green. if it was in red the connection/key was not working. if in orange the no entry yet. only MongoDB is optional. API key and MySQL URL was important.

For this configuration read the text above the entry button. Or you can go with detialed configuration on configuration.md file.

you can also configure the secrets with streamlit secrets file. get how to configure at secret_configuration.md Also, keep your secret seperate and make sure to add it to .gitignore file.

# Home Page

In his page we will extract the data from the Google's YouTube API. You enter multiple Channel names, Channel ID in the entry box. It will automatically get the respective channel id with given channel name. the max limit was set to 10. change that in configuration mention above. 

                My advise is to select the channels with lower playlist,videos, comments. 
                it can processed fast on testing time. And don't exced the free token limit.
In Background of this page the data was stored in JSON file format with date and time in it. So, you can get multiple channels at any given time and process for visualization at any time. More channels means more compute power and More tokens. Google's Free youtube API token was limited to 10,000 only. For more detials visit YouTube documentation.

# Storage page

This page where the converted data was uploaded to MySQL and MongoDB. If you want to proccess with MongoDB click the checkbox. Else leve it empty. And there will be download button shown. where you can download the JSON file for yourself. This helps with Tableau. you can directly process the JSON file. Upon clicking upload button, wait for the process to complete. It will show the time to complete on uploads.

# Analysis page

This Analaysis page gives 10 questions and code for that. upon clicking the data that will gives the result in dataframe. stremlit data was limited to show only one table. so, make the query to display only one table.

# Plot page

This plot page will display charts for comparison for channels, playlist, videos and comments with sentiment. There are multiple charts can be added to this page. but made limited no of charts only. For more you can go with datavisualization tools.
